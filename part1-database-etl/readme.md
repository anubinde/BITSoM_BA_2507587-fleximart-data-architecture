#Part 1-database etl
This is a Star Schema Data Warehouse project built to support business intelligence. It features an ETL Pipeline (ETL_pipeline.ipynb) for data cleaning, transformation, and loading into the fact_sales, dim_customer, and dim_product tables. The warehouse supports advanced OLAP analytics (analytics_queries.sql) and operational reporting (Business_queries.sql).
Setup and Execution
1.ETL: Run ETL_pipeline.ipynb to clean the raw data and populate the Star Schema tables.
2.Analysis: Execute the queries in Business_queries.sql or analytics_queries.sql against the loaded data warehouse.
